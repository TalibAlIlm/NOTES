What is this application?

A premium structured digital notebook for Islamic studies — formatted for A4 printing with authentic geometric ornaments and classical typography.

Write using four color-coded block types: Āyah (Quranic verses), Ḥadīth (prophetic narrations), Athar / Sharḥ (scholarly commentary), and plain Notes. Documents are saved automatically and organized in your personal library.

🔄
Spaced Repetition — Study Smarter, Not Harder

Built-in SM-2 Spaced Repetition — the same algorithm behind Anki. Instead of re-reading everything daily, the system schedules each document at the optimal moment before you forget it.

Important — Your Data & Storage

Your notes are saved in this browser's local storage only.

  • Clearing browser data or cookies will permanently delete all notes.
  • Use Export Backup regularly to download a JSON file of all documents.
  • Use Import Backup to restore — documents merge without duplicates.
  • Notes are tied to this specific browser on this device.
